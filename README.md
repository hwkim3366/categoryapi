JDK  - 11
RDBMS - H2


## 1.설치

    - `$ git clone https://github.com/hwkim3366/categoryapi.git`


## 2.빌드

    - `$ ./gradlew bootJar`


## 3.실행

    - `$ java -jar ./build/libs/category-api-0.0.1-SNAPSHOT.jar`

    
## 4.SWAGGER 접속 후 확인

    - `http://localhost:8080/swagger-ui/`
